# Plan 01-03: Runnable placeholder apps + dev wiring

<objective>
Give each workspace a minimal runnable entry so `npm run dev` from root boots BOTH apps and proves the wiring end-to-end: web serves a Vite placeholder page, server listens on localhost and prints its URL.
</objective>

<must_haves>
  <truths>
    - Root `npm run dev` starts both web and server concurrently
    - Server prints its localhost URL on boot
    - Web serves a placeholder page that imports a shared type (proves workspace linkage)
  </truths>
</must_haves>

<tasks>

<task id="T1" type="execute">
<action>
Create `apps/server/src/main.ts` — minimal Fastify server:
```typescript
import Fastify from 'fastify';
import type { LogEvent } from '@wan22/shared';

const PORT = Number(process.env.PORT) || 7421;
const HOST = '127.0.0.1';

const app = Fastify({ logger: true });

app.get('/api/health', async () => ({ status: 'ok', time: new Date().toISOString() }));

const start = async () => {
  try {
    await app.listen({ port: PORT, host: HOST });
    const url = `http://${HOST}:${PORT}`;
    app.log.info(`Wan2.2Animate Deploy server listening on ${url}`);
    // Prove shared types are linked:
    const sample: LogEvent = {
      timestamp: new Date().toISOString(),
      level: 'info',
      message: 'server bootstrapped',
    };
    app.log.info({ sample }, 'shared types linked');
  } catch (err) {
    app.log.error(err);
    process.exit(1);
  }
};

start();
```
Ensure `apps/server` has `@wan22/shared` in dependencies (already added in 01-01 T3) and `fastify` installed.
</action>
<read_first>
  - apps/server/package.json
  - packages/shared/src/types.ts
</read_first>

</task>

```tsx
import type { InstanceStatus } from '@wan22/shared';

const statuses: InstanceStatus[] = ['idle', 'building', 'ready'];

export function App() {
  return (
    <div style={{ fontFamily: 'system-ui', padding: '2rem' }}>
      <h1>Wan2.2Animate Deploy</h1>

```
- `apps/web/vite.config.ts`:
```typescript
import { defineConfig } from 'vite';
import react from '@vitejs/plugin-react';

export default defineConfig({

  },
});
```
