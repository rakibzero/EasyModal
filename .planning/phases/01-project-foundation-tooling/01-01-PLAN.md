# Plan 01-01: Scaffold monorepo structure

<objective>
Create the npm-workspaces monorepo skeleton: root package.json with workspaces, base tsconfig, per-app packages, and shared lint/format config. End state: `npm install` resolves all workspaces.
</objective>

<must_haves>
  <truths>
    - npm workspaces declared so `apps/*` and `packages/*` are linked
    - TypeScript strict base config that per-app configs extend
    - Shared ESLint + Prettier config at root
  </truths>
</must_haves>

<tasks>

<task id="T1" type="execute">
<action>
Create root `package.json`:
```json
{
  "name": "wan22-animate-deploy",
  "version": "0.1.0",
  "private": true,
  "type": "module",
  "workspaces": ["apps/*", "packages/*"],
  "scripts": {
    "dev": "concurrently -n web,server -c blue,green \"npm -w apps/web run dev\" \"npm -w apps/server run dev\"",
    "build": "npm -w packages/shared run build && npm -w apps/web run build && npm -w apps/server run build",
    "lint": "eslint . --ext .ts,.tsx",
    "format": "prettier --write .",
    "typecheck": "tsc -b"
  },
  "devDependencies": {
    "concurrently": "^9.0.0",
    "eslint": "^9.0.0",
    "prettier": "^3.3.0",
    "typescript": "^5.6.0",
    "typescript-eslint": "^8.0.0",
    "@typescript-eslint/eslint-plugin": "^8.0.0",
    "@typescript-eslint/parser": "^8.0.0"
  }
}
```
</action>
<read_first>
  - .planning/PROJECT.md (decisions: npm workspaces, MIT)
  - .planning/PRODUCTION-PLAN.md (§3 file structure)
</read_first>
<acceptance_criteria>
  - `package.json` contains `"workspaces": ["apps/*", "packages/*"]`
  - `package.json` contains scripts: dev, build, lint, format, typecheck
  - `package.json` has `concurrently` and `typescript` in devDependencies
</acceptance_criteria>
</task>

<task id="T2" type="execute">
<action>
Create `tsconfig.base.json` (strict, ES2022, moduleResolution Bundler):
```json
{
  "compilerOptions": {
    "target": "ES2022",
    "module": "ESNext",
    "moduleResolution": "Bundler",
    "strict": true,
    "esModuleInterop": true,
    "skipLibCheck": true,
    "forceConsistentCasingInFileNames": true,
    "resolveJsonModule": true,
    "declaration": true,
    "declarationMap": true,
    "sourceMap": true
  }
}
```
Then per-app tsconfigs (`apps/web/tsconfig.json`, `apps/server/tsconfig.json`, `packages/shared/tsconfig.json`) each extending `../../tsconfig.base.json` with appropriate `outDir`, `rootDir`, and (for web) `"jsx": "react-jsx"`.
</action>
<read_first>
  - tsconfig.base.json (after T1 creates it — actually independent)
</read_first>
<acceptance_criteria>
  - `tsconfig.base.json` contains `"strict": true`
  - `apps/web/tsconfig.json` contains `"extends": "../../tsconfig.base.json"` and `"jsx": "react-jsx"`
  - `apps/server/tsconfig.json` and `packages/shared/tsconfig.json` each contain `"extends": "../../tsconfig.base.json"`
</acceptance_criteria>
</task>

<task id="T3" type="execute">
<action>
Create per-workspace `package.json` files:
- `packages/shared/package.json`: name `@wan22/shared`, main `./dist/index.js`, types `./dist/index.d.ts`, scripts build (`tsc`), type module, dependencies none.
- `apps/server/package.json`: name `@wan22/server`, private true, scripts dev (`tsx watch src/main.ts`), build (`tsc`), dependencies `fastify`, `@wan22/shared`, devDependencies `tsx`, `@types/node`, `typescript`.
- `apps/web/package.json`: name `@wan22/web`, private true, scripts dev (`vite`), build (`tsc -b && vite build`), preview (`vite preview`), dependencies `react`, `react-dom`, `@wan22/shared`, devDependencies `vite`, `@vitejs/plugin-react`, `typescript`, `@types/react`, `@types/react-dom`.

Create shared ESLint flat config `.eslintrc.cjs` (or `eslint.config.js` for ESLint 9 flat config) applying typescript-eslint recommended, ignoring `dist`/`node_modules`. Create `.prettierrc.json` with `{ "semi": true, "singleQuote": true, "trailingComma": "all", "printWidth": 100 }`.
</action>
<read_first>
  - package.json (root)
</read_first>
<acceptance_criteria>
  - `packages/shared/package.json` name is `@wan22/shared`
  - `apps/server/package.json` name is `@wan22/server` and lists `fastify` and `@wan22/shared` in dependencies
  - `apps/web/package.json` name is `@wan22/web` and lists `react`, `react-dom`, `@wan22/shared`
  - `.prettierrc.json` contains `"singleQuote": true`
  - ESLint config file exists and references typescript-eslint
</acceptance_criteria>
</task>

</tasks>
